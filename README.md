# Successor and Predecessor Calculator

This is a simple Python exercise that calculates the successor and predecessor of a given number.

## How it works

The user inputs a number, and the program returns:
- The successor (number + 1)
- The predecessor (number - 1)

## Example

Input:
10

Output:
The successor is 11 and the predecessor is 9

## Purpose

This exercise was created to practice basic programming concepts such as:
- Variables
- User input
- Arithmetic operations
